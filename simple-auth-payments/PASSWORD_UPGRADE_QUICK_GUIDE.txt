WHAT IT DOES
═══════════════════════════════════════════════════════════════

Automatically upgrades weak password hashes to strong ones during
login, without requiring users to reset their passwords.

Old hash: $2b$04$... (4 rounds - weak)
          ↓ (automatic upgrade during login)
New hash: $2b$12$... (12 rounds - strong)

═══════════════════════════════════════════════════════════════
HOW IT WORKS
═══════════════════════════════════════════════════════════════

1. User logs in with their password
2. System verifies password (works with old hash)
3. System detects hash is weak (< 12 rounds)
4. System generates new strong hash (12 rounds)
5. System saves new hash to database
6. User is logged in successfully
7. Next login uses the strong hash
